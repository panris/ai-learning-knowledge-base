## 🤔 什么是 GPT Image 2？

**GPT Image 2**（代号 **"duct-tape"**）是 OpenAI 下一代图像生成模型。社区测试反馈它在以下方面实现了质的飞跃：

- 🎯 **像素级文字渲染** — 中文、英文、日文均达到 native 水准，无错字、无字形扭曲
- 🎨 **跨图像素级一致性** — 同一角色、风格、IP 在多张图间保持像素级一致
- ⚡ **商用级插画质量** — 插画风格输出无需人工精修，即可直接用于商业场景
- 🌈 **真实艺术风格注入** — 不止是"模仿参考图"，而是真正理解并再现艺术风格的灵魂
- 🔧 **故事板与产品系列** — 适合故事板、IP 形象、产品系列图等需要多图一致性的场景
- 📐 **多语言平面设计** — 社交卡片、Banner、海报一次生图即可完成多语言文字排版

## 💡 使用技巧

### 1. 结构化提示词格式

GPT Image 2 支持 JSON 格式的结构化提示词，便于精确控制：

```json
{
  "type": "图像类型",
  "subject": "主体描述",
  "style": "风格设定",
  "background": "背景描述",
  "layout": {
    "centerpiece": "中心元素",
    "sections": []
  }
}
```

### 2. 动态参数（Raycast Friendly）

部分提示词支持动态参数，可快速替换：

```
{argument name="参数名" default="默认值"}
```

**示例：**
```
生成一张 {argument name="city name" default="成都"} 的城市地图
```

### 3. 多图一致性技巧

- 使用相同的人物描述
- 保持风格关键词一致
- 指定相同的光影条件
- 使用参考图（if supported）

### 4. 文字渲染要点

- 明确指定字体风格
- 使用引号包裹需要渲染的文字
- 指定文字位置（居中、左上角等）
- 说明文字大小和颜色

### 5. 提示词优化建议

| 元素 | 建议 |
|------|------|
| 主体 | 清晰描述主体特征（颜色、形状、位置） |
| 风格 | 使用具体风格关键词（水彩、赛博朋克、像素艺术） |
| 光影 | 指明光源方向和强度 |
| 构图 | 说明视角（俯视、仰视、平视） |
| 细节 | 添加纹理、材质描述 |
| 氛围 | 使用情绪关键词（温馨、神秘、紧张） |
