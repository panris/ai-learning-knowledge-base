## 一、什么是提示词？

提示词（Prompt）就是您给 AI 的"指令"。同样一个问题，问法不同，AI 给出的答案质量天差地别。

**差的提问：**
> 帮我写文章

**好的提问：**
> 请帮我写一篇公众号文章，主题是"如何保持好心情"，读者是50-65岁的中老年人，语言要亲切自然，字数800字左右，分3-4个小节，每节有小标题。

---

## 二、提示词黄金公式

```
角色 + 任务 + 背景 + 要求 + 格式
```

| 要素 | 说明 | 示例 |
|------|------|------|
| 角色 | 让 AI 扮演什么角色 | "你是一位经验丰富的营养师" |
| 任务 | 具体要做什么 | "帮我制定一份健康食谱" |
| 背景 | 相关背景信息 | "我今年60岁，有轻度高血压" |
| 要求 | 具体限制条件 | "低盐低脂，食材要常见易买" |
| 格式 | 输出格式要求 | "按早中晚分列，表格形式" |

## 三、实用提示词模板

### 📝 写作类

**公众号文章**
```
请写一篇公众号文章，主题：[主题]
目标读者：[读者群体]
文章风格：[亲切/严肃/幽默]
字数：[字数]
要求：有吸引人的开头，分小节，结尾有互动引导
```

**朋友圈文案**
```
帮我写3条朋友圈文案，关于[主题]
风格：[温暖/励志/生活化]
字数：每条不超过100字
加上合适的emoji
```

**读书笔记**
```
我读了《[书名]》，请帮我整理读书笔记
包括：核心观点、金句摘录、个人感悟
格式：分点列出，简洁易读
```

---

### 💬 日常对话类

**解释复杂概念**
```
请用大白话解释[概念]，
就像给一个完全不懂的老人解释一样，
举一个生活中常见的例子
```

**翻译润色**
```
请将以下文字翻译成[语言]，
要求：自然流畅，符合当地表达习惯
[原文]
```

**总结归纳**
```
请帮我总结以下内容的核心要点，
用3-5条列出，每条不超过30字：
[内容]
```

---

### 🏥 生活实用类

**健康咨询**
```
我[年龄]岁，有[症状/情况]，
请给我一些[饮食/运动/生活]方面的建议，
注意：我还有[其他健康状况]
（重要：仅供参考，具体请咨询医生）
```

**旅行规划**
```
帮我规划一次[目的地]的旅行
出发地：[城市]
时间：[天数]
人员：[人数和年龄]
预算：[大概金额]
偏好：[休闲/文化/美食]
```

## 四、提示词进阶技巧

### 技巧1：给 AI 一个身份
> ❌ "帮我写一封信"
> ✅ "你是一位文笔优美的老师，帮我写一封感谢信"

### 技巧2：提供例子
> "请按照下面这个风格写：[示例文字]"

### 技巧3：分步骤要求
> "请先列出大纲，我确认后再写正文"

### 技巧4：要求检查
> "写完后，请检查是否有错别字，语句是否通顺"

### 技巧5：迭代优化
> "这个不错，但请把语气改得更温和一些"
> "第二段太长了，请精简到100字以内"
