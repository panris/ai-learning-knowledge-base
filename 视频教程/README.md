## 七、学习建议

1. **先看入门视频**（1-2小时）建立整体认知
2. **边看边动手** - 打开 AI 工具跟着操作
3. **做笔记** - 记录好用的提示词和技巧
4. **加入社群** - 遇到问题有人交流
