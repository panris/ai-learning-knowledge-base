# 🌱 基础入门

> 从零开始学 AI，不需要任何基础，看完这里你就入门了。

---

## 一、什么是 AI？

人工智能（Artificial Intelligence，简称 AI），简单说就是让机器像人一样"思考"和"学习"。

现在我们日常用到的 AI，主要分几类：

| 类型 | 例子 | 能做什么 |
|------|------|----------|
| 对话 AI | ChatGPT、文心一言 | 聊天、写作、答疑 |
| 图像 AI | Midjourney、Stable Diffusion | 画图、修图、生成图片 |
| 语音 AI | 讯飞语音、Whisper | 语音转文字、文字转语音 |
| 视频 AI | Sora、可灵 | 生成视频、视频剪辑 |
| 代码 AI | GitHub Copilot、Cursor | 辅助编程 |

## 二、入门学习路线

### 第一阶段：了解 AI（1-2周）
- [ ] 了解 AI 的基本概念
- [ ] 注册并体验 ChatGPT 或文心一言
- [ ] 学会基本的提问方式（提示词）
- [ ] 了解 AI 能做什么、不能做什么

### 第二阶段：上手工具（2-4周）
- [ ] 掌握 ChatGPT / Claude 的使用技巧
- [ ] 学会用 AI 辅助写作、翻译、总结
- [ ] 尝试 AI 画图工具（Midjourney 或国产工具）
- [ ] 探索 AI 在自己工作中的应用场景

### 第三阶段：深入学习（1-3个月）
- [ ] 学习提示词工程（Prompt Engineering）
- [ ] 了解大模型的基本原理
- [ ] 尝试用 AI 做一个小项目
- [ ] 关注 AI 行业动态

## 五、常见问题

**Q：学 AI 需要会编程吗？**
A：不需要！大多数 AI 工具直接用就行，不需要写代码。如果想深入研究，学一点 Python 会有帮助。

**Q：AI 会取代我的工作吗？**
A：AI 会改变很多工作方式，但更多是辅助而非替代。学会用 AI，反而能让您更有竞争力。

**Q：国内能用 ChatGPT 吗？**
A：需要特殊网络环境。国内可以用文心一言、通义千问、讯飞星火等替代产品，功能也很强大。

**Q：AI 说的话都是真的吗？**
A：不一定！AI 有时会"一本正经地胡说八道"，重要信息一定要核实。
