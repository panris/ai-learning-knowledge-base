## 一、必读经典论文

### Transformer（2017）

**标题**：Attention Is All You Need

**作者**：Google Brain 团队

**意义**：奠定了现代大语言模型的基础架构

**核心创新**：
- 提出 Transformer 架构
- 引入 Self-Attention 机制
- 抛弃了传统的 RNN/LSTM

**中文解读**：[Transformer 详解](https://zhuanlan.zhihu.com/p/44121378)

---

### GPT 系列

#### GPT-1（2018）
**标题**：Improving Language Understanding by Generative Pre-Training

**意义**：首次证明"预训练+微调"范式的有效性

#### GPT-2（2019）
**标题**：Language Models are Unsupervised Multitask Learners

**意义**：展示了大模型的涌现能力

#### GPT-3（2020）
**标题**：Language Models are Few-Shot Learners

**意义**：1750亿参数，开启大模型时代

#### GPT-4（2023）
**标题**：GPT-4 Technical Report

**意义**：多模态能力，接近人类水平

---

### BERT（2018）

**标题**：Pre-training of Deep Bidirectional Transformers

**作者**：Google

**意义**：双向编码器，理解能力更强

**应用**：搜索、问答、文本分类

## 五、论文阅读建议

### 阅读顺序

1. **先看综述** - 了解领域全貌
2. **读经典** - Transformer、GPT、BERT
3. **追最新** - 关注 arXiv 每日更新
4. **看解读** - 中文博客、视频讲解

### 推荐资源

| 资源 | 链接 | 说明 |
|------|------|------|
| arXiv | arxiv.org/list/cs.AI/recent | 最新论文 |
| Papers With Code | paperswithcode.com | 论文+代码 |
| 知乎专栏 | 搜索"AI论文解读" | 中文解读 |
| 李沐论文精读 | B站搜索 | 视频讲解 |

### 高效阅读技巧

1. **先看摘要** - 判断是否需要细读
2. **看图和表** - 快速理解核心方法
3. **读结论** - 了解主要贡献
4. **再读方法** - 深入理解实现细节
