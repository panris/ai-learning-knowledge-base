# 🔧 飞书多维表格 AI 咒语大全

> 用自然语言指挥飞书，自动生成表结构、自动填充、自动分类、自动汇总

---

## 一、基础认知

### 1.1 飞书 AI 能做什么

| 能做 ✅ | 不能做 ❌ |
|---------|----------|
| 根据描述自动生成字段结构 | 读取已有数据（需先有表） |
| 自动填充字段内容（AI 写内容） | 直接生成图表（需先有数据） |
| 根据条件自动标记/分类 | 修改其他人的权限设置 |
| 自动生成摘要、提取关键词 | 实时同步外部数据 |
| 对已有字段进行自然语言查询 | 批量上传文件到附件字段 |

### 1.2 两种触发方式

**方式一：自动字段（推荐）**
> 字段类型 → 选「自动」类别 → 选「AI 字段」→ 选择具体能力

**方式二：自然语言搜索**
> 在多维表格顶部搜索框 → 直接用自然语言提问 → AI 理解后筛选/返回结果

## 二、字段生成咒语

### 模板 A：员工信息表
```
帮我建一个员工信息表，需要包含：
姓名、部门、入职日期、转正状态、职级、直接汇报对象、紧急联系人、电话号码、试用期剩余天数。

### 模板 B：项目管理表
```
帮我建一个项目进度管理表，包含：
项目名称、项目负责人、开始时间、截止时间、当前阶段（启动/执行/收尾）、完成度百分比、风险等级（高/中/低）、主要阻碍、下一步行动。
```

### 模板 C：客户跟进表
```
帮我建一个客户跟进记录表，包含：
客户名称、客户类型、对接销售、首次联系日期、最近跟进日期、下次跟进时间、意向等级（A/B/C）、跟进摘要、最近一次跟进方式（电话/微信/面谈）。
```

## 三、内容填充咒语

### 模板 D：批量生成内容
```
在「产品名称」字段中，为我填充 20 条真实的智能硬件产品名称，包括手环、手表、音箱、耳机品类。

```
在「项目简介」字段中，根据「项目名称」和「项目目标」两个字段的内容，为每个项目生成一段不超过 50 字的项目简介。
```

### 模板 F：拆解待办
```
在「待办事项」字段中，根据「本周目标」字段的内容，拆解出 3-5 条具体的待办事项，每条不超过 20 字，用 bullet point 格式列出。
```

## 四、自动分类咒语

### 模板 G：自动评级
```
在「意向等级」字段中，根据「最近跟进日期」和「跟进摘要」两个字段，判断该客户的意向等级：
- 3 天内有跟进且摘要积极 → A（高意向）
- 7 天内有跟进 → B（中意向）
- 超过 7 天未跟进 → C（低意向）
```

### 模板 H：自动打标签
```
在「风险标签」字段中，根据「当前阶段」和「完成度百分比」判断风险：
- 截止日期在 7 天内且完成度低于 50% → 「红色高风险」
- 截止日期在 7 天内且完成度 50-80% → 「黄色中风险」
- 其他 → 「绿色正常」
```

### 模板 I：自动分类
```
在「合同类型」字段中，根据「客户名称」和「签约金额」判断：
- 签约金额 ≥ 100 万 → 「战略客户」
- 签约金额 20-100 万 → 「重点客户」
- 签约金额 < 20 万 → 「标准客户」
```

## 七、实战案例：员工入职进度管理表

### 目标
用自然语言，10 分钟内搭完一套完整的入职进度管理表。

### Step 1：建表
```
帮我建一个员工入职进度管理表，包含：
员工姓名、入职日期、试用期截止日期、当前入职阶段（Offer确认/资料提交/入职培训/试用中/已转正）、资料完成度（百分比）、培训完成状态（未开始/进行中/已完成）、入职手续办理状态、HR对接人、备注。
```

### Step 2：批量填充示例数据
```
在「员工姓名」字段填充 10 个常见的中国姓名。
在「入职日期」字段，填充从今天起往前 1-6 个月不等的日期。
```

### Step 3：自动评级
```
根据「资料完成度」和「培训完成状态」，自动判断入职进度是否正常：
- 资料完成度 100% 且培训已完成 → 「入职流程完成」
- 资料完成度 50-99% → 「入职流程中」
- 其他 → 「入职流程滞后」
```

### Step 4：自动预警
```
在「HR提醒」字段中，判断：试用期截止日期在 30 天内的，提示「请准备转正评估」；在 7 天内的提示「⚠️ 紧急：本周内完成转正手续」。
```

## 八、常见错误与修正

### 错误一：AI 填充的内容不对
**修正咒语**：
```
请重新填充「XX」字段，这次注意：
1. 只写[具体要求]
2. 不要包含[不需要的内容]
3. 格式必须是[具体格式]
```

### 错误二：公式报错了
在咒语末尾加上：
```
提示：飞书多维表格公式语法与 Excel 略有不同，请使用 IF(条件, 是, 否) 结构，字段引用用 [字段名]。
```

### 错误三：分类结果全是一个值
**修正咒语**：
```
重新判断「XX」字段，严格按以下规则：
- [具体条件A] → 输出「A」
- [具体条件B] → 输出「B」
- 不满足以上条件 → 输出「其他」
```
