# 🎬 对比实验视频 · 分镜脚本

**片名**：同样是问 AI，差距在哪里？
**时长**：90 秒
**格式**：分屏演示（左侧 Prompt / 右侧 AI 反馈）
**旁白**：亲和力强的 AI 人声

---

## 分镜总览

| 段落 | 时长 | 类型 | 内容 |
|------|------|------|------|
| 钩子 Hook | 0–8s | 口播 | 抛出问题，引发好奇 |
| 对比实验 A | 8–35s | 分屏 | 模糊提问 → AI 敷衍回答 |
| 对比实验 B | 35–65s | 分屏 | 公式化提问 → AI 高质量回答 |
| 揭晓答案 | 65–78s | 口播 | 解释 ABCD 公式为什么有效 |
| 行动号召 | 78–90s | 口播+字幕 | 教一句话，立刻用起来 |

---

## 详细分镜

### 🎬 段落 1：钩子（0:00–0:08）

**画面**：真人 / 数字人正面，背景简洁白色或淡色渐变，中间字幕叠加

**口播文案**：
> "你发一条消息给 AI，对面那个 AI，水平忽高忽低、回答时好时坏——你以为它是随机的，其实它是跟着你的提问走的。"

**字幕叠加**：
```
❌ 模糊提问  →  AI 随机发挥
✅ 公式提问  →  AI 精准输出
```

---

### 🎬 段落 2：对比实验 A — 模糊提问（0:08–0:35）

**画面**：分屏，左侧演示 AI 提问界面，右侧演示 AI 输出界面

#### 镜头 A-1：输入 Prompt
**左侧**（打字动画）：
```
请帮我写一封邮件
```

#### 镜头 A-2：AI 输出
**右侧**（打字动画）：
> "您好。感谢您的来件。我已收到您的邮件，关于您提到的内容，我有一些想法，但需要更多信息才能给出具体建议。建议您提供更多细节，我可以进一步帮助您。"

**口播**：
> "好，这条邮件，用不上。说了等于没说，还要继续问。这就是模糊提问的结果——AI 只能给你一个通用模板。"

#### 镜头 A-3：问题标注
```
❌ 语气太泛，没针对性
❌ 没有说明邮件目的是什么
❌ 没有任何格式和结构
❌ 没法直接用，还要继续改
```

---

### 🎬 段落 3：对比实验 B — 公式化提问（0:35–1:05）

**画面**：分屏，风格与段落 A 完全一致

#### 镜头 B-1：输入 Prompt（ABCD 公式）
**左侧**（打字动画）：
```
你是一位资深销售经理。
帮我写一封求职拒绝邮件。
候选人等了3周，资历优秀但HC已冻结，语气专业友好。
正文200字以内，包含感谢、拒绝原因、后续合作意愿，结尾祝好。
```

#### 镜头 B-2：AI 输出
**右侧**（打字动画）：
> "尊敬的[候选人姓名]：
>
> 非常感谢您参加我们的招聘流程。您的资历非常出色，但目前公司HC已冻结，无法继续推进本次招聘。期待未来有机会再次合作。祝您前程似锦！"

**口播**：
> "直接能用，对不对？"

#### 镜头 B-3：优点标注
```
✅ 语气专业、换位思考
✅ 理由具体（HC冻结）
✅ 结构清晰：感谢 → 拒绝 → 祝福
✅ 字数控制在了要求范围内
✅ 改个名字就能发
```

---

### 🎬 段落 4：揭晓答案（1:05–1:18）

**字幕逐行出现**（居中大字）：

```
A — 角色：你希望 AI 扮演谁？
B — 任务：需要 AI 完成什么？
C — 背景：有什么必要信息要告诉它？
D — 要求：输出格式和风格有什么限制？
```

**口播**：
> "ABCD 公式，就这四步。把你的问题装进这个框架里，AI 就知道你到底要什么。"

---

### 🎬 段落 5：行动号召（1:18–1:30）

**口播**：
> "现在就试一下。把你要问 AI 的那句话，用 ABCD 重新说一遍。你会发现——同一个 AI，这次完全不同。"

**最后画面**：
```
把 AI 当人看
学会下命令
```

## 制作说明

### 拍摄 / 录制建议

| 要素 | 说明 |
|------|------|
| 设备 | 手机竖屏 4K 即可 |
| 分屏效果 | 剪映 App「分屏」模板，左侧放提问截图，右侧放 AI 回答截图 |
| 打字动画 | 剪映「打字机」效果，语速调至 1.5x |
| AI 界面录屏 | 用飞书 AI / ChatGPT 实时录屏，提前准备好提问和输出 |
| 背景音乐 | 轻爵士或钢琴，音量低于人声 30% |

### 旁白录制建议
- 推荐工具：**剪映**（文字转语音）→ 选择女声，语速 1.2x，音调 +5
- 或用 **ElevenLabs** / **微软 Azure TTS** 生成更自然人声
