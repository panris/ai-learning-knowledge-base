# 📋 50 场景万能指令库

> 复制即用，换关键词质量不变

---

## 框架说明

所有指令遵循 **ABCD 公式**：

| 要素 | 说明 |
|------|------|
| **A 角色** | 你希望 AI 扮演什么身份？ |
| **B 任务** | 需要 AI 完成什么？ |
| **C 背景** | 补充必要的背景信息 |
| **D 要求** | 指定输出的格式与风格 |

## 工具适配

所有指令均兼容主流 AI 工具，部分场景有最佳推荐：

| 场景 | 通用推荐 | 最佳推荐 |
|------|---------|---------|
| 行政（邮件/通知/流程） | 豆包、通义千问 | 飞书 AI（企业协作） |
| 财务（数据分析/报告） | ChatGPT、Claude | Kimi（可上传表格） |
| 运营（内容/数据/竞品） | ChatGPT、通义千问 | 豆包（中文内容强） |
| 生活（规划/建议/辅导） | 豆包、文心一言 | 任意即可 |
| 学习（翻译/润色/讲解） | Claude、ChatGPT | Gemini（跨语言强） |

> 💡 不确定用哪个？先用豆包，免费且中文体验最好。详见 [AI工具大全](../../AI工具大全/README.md)

## 行政场景（01–10）

### 01 · 会议邀请邮件

```
你是一位资深行政专员。

帮我写一封会议邀请邮件。

背景：下周三下午3点，公司第一会议室，内容是Q2季度复盘，参会者包括华东区全体销售。

要求：语气正式但不刻板，邮件正文包含会议主题、时间、地点、议程概览四个部分，正文不超过300字。
```

### 02 · 会议室预约通知

```
你是一位行政助理。帮我在邮件正文中写一段会议室预约确认信息。

背景：已预约"上海厅"给财务部，4月25日（周五）14:00-17:00，用于年度审计沟通会，已与行政前台确认档期。

要求：格式为确认通知，语气友好，注明会议室名称、时间、用途、部门。
```

### 03 · 内部公告

```
你是一位企业品牌专员。帮我起草一份内部公告。

背景：公司将从5月1日起实行新的弹性工作制，允许每周一天居家办公，需告知全体员工并说明申请流程。

要求：公告格式，分三段：制度说明、申请方式、生效时间。语言简洁清晰，避免使用法律术语。
```

### 04 · 来访接待流程

```
你是一位行政接待主管。帮我就以下场景写一段接待标准话术。

背景：重要客户明天来访，参观时长约2小时，包含产品演示和商务洽谈环节，我方接待团队共3人。

要求：包含开场问候、参观动线简介、茶歇安排三个模块，每模块不超过80字。
```

### 05 · 节假日值班通知

```
你是一位行政人事专员。帮我起草五一假期值班安排通知。

背景：公司五一假期为5月1日-5月5日，行政部门每天安排1人在岗，值班地点在公司前台，值班时间9:00-17:00。

要求：通知正文包含值班日期、值班人员姓名、联系电话、紧急联系人，格式对齐工整。
```

### 06 · 名片内容优化

```
你是一位品牌文案专家。帮我优化以下名片文字，使其更专业有辨识度。

背景：我是一名独立保险经纪人，从业8年，专注高净值客户的家庭资产配置和财富传承。

要求：提炼成一句不超过20字的个人价值主张，另附一句不超过15字的专业领域描述。
```

### 07 · 合同条款通俗说明

```
你是一位法务行政助理。帮我为以下合同条款写一段通俗化说明，供非法律专业的业务部门同事理解。

背景：合同中的知识产权归属条款："乙方在本项目执行过程中产生的所有智力成果，其知识产权归甲方永久享有"。

要求：用大白话分两点解释：1）这条款对乙方意味着什么；2）签约前需要特别注意什么。
```

### 08 · 办公室搬迁通知

```
你是一位行政项目经理。帮我起草一份办公室搬迁内部通知。

背景：公司将于6月15日（周日）从A座12楼搬迁至B座8楼，搬迁当天全体在家办公，6月16日起在新地址上班。

要求：通知包含搬迁时间节点、新地址、停车指引、当天安排四个部分，语言温和积极。
```

### 09 · 报销流程指南

```
你是一位财务行政专员。帮我写一份报销流程指南，面向刚入职的新员工。

背景：公司使用钉钉进行费用报销，单笔报销超过2000元需要部门总监审批，不超过2000元由财务专员审批，每月25日统一结算。

要求：分步骤说明，共5步，每步一句话，语言通俗，附一个常见Q&A。
```

### 10 · 供应商比选报告

```
你是一位采购行政专员。帮我写一份三家供应商的比选摘要报告。

背景：需要采购20台办公笔记本电脑，三家供应商分别为联想（价格适中，售后好）、戴尔（同配置价格高15%，品牌更高端）、惠普（价格最低，交付需3周）。

要求：用表格对比三家供应商的"价格"、"交付时间"、"售后服务"、"推荐理由"四个维度，最终给出采购建议。
```

## 财务场景（11–20）

### 11 · 催款函

```
你是一位财务专员。帮我写一封客户催款函。

背景：客户公司应付账款已逾期30天，欠款金额128,000元，合同约定付款日为3月15日，今日为4月14日，已发送一次友好提醒。

要求：语气专业但有力度，开头说明欠款金额和逾期天数，中间说明合同依据，结尾明确还款期限（7个工作日内）并说明后续措施。
```

### 12 · 月度财务分析摘要

```
你是一位财务分析师。帮我写一份本月财务表现摘要，面向公司管理层。

背景：本月营收380万元，环比增长12%，成本280万元，环比增长8%，净利润100万元，环比增长23%。毛利率约26%，上期为22%。

要求：摘要结构为：整体结论 → 关键数据对比 → 主要变化原因（2-3点）→ 下月关注事项。语言精炼，每部分不超过3句话。
```

### 13 · 预算申请说明

```
你是一位部门财务BP。帮我写一段部门下半年预算申请说明。

背景：市场部申请增加线下活动预算15万元，理由是下半年计划举办3场行业沙龙，每场预算5万元，预计可带来150个高质量线索。

要求：说明申请金额、用途、时间计划、预期效果（量化），语气理性客观，适合提交给财务总监审批。
```

### 14 · 发票异常处理通知

```
你是一位财务专员。帮我就发票异常情况写一封内部协作邮件。

背景：某供应商4月8日开具的增值税专用发票，税额标注有误，应为13%税率实为6%，财务无法入账，需要供应商重新开具。

要求：邮件包含：问题描述（发票号、金额）、错误详情、处理要求（重新开具时间、寄送方式）、联系人信息。
```

### 15 · 费用管控建议

```
你是一位财务总监。帮我起草一份全公司差旅费用管控建议。

背景：本季度差旅费用同比增长35%，其中机票和酒店费用占比最大，主要原因是员工习惯直接预订高价商务舱和五星级酒店，缺乏事前比价环节。

要求：建议分三部分：现状描述、管控措施（3条）、预期效果。措施具体可执行，不用抽象原则。
```

### 16 · 付款账期说明

```
你是一位财务专员。帮我写一段付款账期说明，发给需要了解付款节奏的长期合作供应商。

背景：我公司付款周期为月结30天，即每月发生的费用在次月25日统一付款，节假日顺延。

要求：用供应商能理解的语言解释，字面清晰，不留歧义，附一个示例（假设3月5日交货，何时能收到款项）。
```

### 17 · 年度审计材料清单

```
你是一位财务专员。帮我写一份内部审计材料准备清单，供业务部门同事使用。

背景：公司将于5月进行年度审计，审计团队需要各部门提供过去一年的合同、付款记录、审批邮件、发票原件等材料。

要求：清单按部门分类，每项材料注明"必须提供"或"如有则提供"，格式为可打印的核对清单。
```

### 18 · 固定资产折旧解释

```
你是一位财务分析师。帮我写一段通俗解释，说明什么是固定资产折旧，以及它如何影响利润表。

背景：我需要向不懂财务知识的销售团队负责人解释：为什么购买一台30万的服务器后，每个月利润会减少约8,300元。

要求：用类比的方式解释（如"就像租车而不是买车"），分三步说明，最终算出月均折旧金额并给出简单类比。
```

### 19 · 银行对账差异说明

```
你是一位出纳专员。帮我写一份银行账与公司账差异说明，供财务经理审阅。

背景：公司银行日记账余额1,285,000元，银行对账单余额1,301,500元，差异16,500元。经核查：银行已扣手续费1,500元（公司未入账），另有一笔15,000元转账对方尚未确认。

要求：用专业但不复杂的方式说明差异来源、金额、调整建议，最终调整后余额一致。
```

### 20 · 产品成本分析报告

```
你是一位财务分析师。帮我写一份产品成本分析报告摘要。

背景：A产品单位成本结构：原材料48%、人工28%、制造费用14%、包装物流10%。原材料中芯片成本占35%。本周芯片价格上涨20%，需评估对A产品毛利率的影响。

要求：先说明当前毛利率基准，再测算芯片涨价后的新毛利率（给出计算过程），最后给出是否需要调整售价的建议。
```

## 运营场景（21–30）

### 21 · 用户反馈周报摘要

```
你是一位用户运营专员。帮我把本周收集到的200条用户反馈整理成一份结构化摘要。

背景：本周反馈来源包括App内反馈（120条）、客服工单（60条）、微博评论（20条）。反馈集中度最高的三类问题是：登录异常（30%）、账单计费不准（25%）、新功能不会用（20%）。

要求：摘要包含：总体满意度评分、TOP3问题（每个问题说明现象、用户影响、建议对策）、本周亮点反馈一条。
```

### 22 · 活动复盘报告

```
你是一位活动运营专员。帮我写一份上周线下沙龙的复盘报告。

背景：活动名称"品牌焕新发布会"，到场186人，预算8万元，实际花费7.6万元。签到转化率92%，活动满意度均分4.3/5。最大的执行失误是投影设备延迟到场35分钟。

要求：复盘结构：数据概览 → 目标达成情况 → 做得好的一件事 → 需要改进的一件事 → 下次执行的三个行动项。
```

### 23 · 新手引导流程设计

```
你是一位产品运营专员。帮我设计一个新用户注册后的7日引导任务体系。

背景：我们的产品是一款项目管理工具，目标用户是30-50人的中小型创业团队，用户痛点是"买了不会用，用了不坚持"。

要求：按D1到D7每天一个核心任务，每个任务说明：任务名称、完成动作、预期收获。任务难度递进，总时长控制在每天5分钟以内。
```

### 24 · 社群运营话术库

```
你是一位社群运营专员。帮我为产品用户社群写一套日常运营话术模板。

背景：社群目的是提升用户活跃度和付费转化，群内有新手用户和付费老用户，群规模500人，日常需要处理的场景包括：欢迎新人入群、引导用户提问、推送产品更新、邀请付费升级、处理用户投诉。

要求：每个场景写一条话术，语气自然亲切，不像机器人发的，每条不超过100字。
```

### 25 · 内容选题规划

```
你是一位内容运营专员。帮我就以下产品信息规划6月份的内容选题。

背景：产品是一款面向企业客户的在线会议工具，主要功能包括：会议录制、AI会议纪要、屏幕共享、会议转译。目标读者是有远程协作需求的企业用户。

要求：按周规划，每周2个选题，共12个选题。每个选题包含：标题、内容方向、适合发布平台、预计字数。
```

### 26 · 竞品对比分析

```
你是一位运营分析师。帮我写一份竞品对比分析，供内部产品讨论使用。

背景：我司产品是一款CRM系统，对标竞品A（功能全但价格贵，5万/年）、竞品B（价格低但功能少，免费版够用）。目标用户是50人以下的销售团队。

要求：从功能完整度、易用性、价格、售后支持四个维度对比，结论给出"各产品适合什么类型的团队"的建议。
```

### 27 · 用户分层运营策略

```
你是一位用户增长专员。帮我为一款付费在线课程平台设计用户分层运营策略。

背景：平台有学员1万人，按付费金额分为：高付费用户（前5%，年消费5000元以上）、普通付费用户（15%，年消费500-5000元）、免费用户（80%）。各层用户的核心诉求不同。

要求：针对三个用户层各设计一套运营策略，包含：激活策略、转化策略、防流失措施。
```

### 28 · 客服话术优化

```
你是一位客服运营主管。帮我优化一段用户退订处理话术。

背景：用户来电要求取消订阅年度会员，理由是"用不上"。当前标准话术是：请确认您的账号信息，告知退款政策为"7天内全额，30天内按比例"。

要求：新的话术要体现"挽留但不强求"的态度，包含：挽留钩子、替代方案（降级到月度会员）、情感收尾。
```

### 29 · 数据看板解读

```
你是一位数据运营专员。帮我写一份本周核心数据看板解读，发给运营团队负责人。

背景：本周关键数据——DAU 8,200（上周8,500），新注册用户340（上周280），付费转化率3.1%（上周2.8%），ARPU 28元（上周26元）。

要求：数据解读要说明：哪些数据值得关注、变化原因假设、每周需要重点观察的指标。
```

### 30 · 业务流程SOP

```
你是一位运营经理。帮我写一份电商订单处理的标准操作流程（SOP）。

背景：订单从用户付款成功到发货，需要经过：订单确认 → 库存核查 → 仓库配货 → 快递下单 → 物流跟踪五个步骤。异常情况包括：库存不足、地址不详、支付异常。

要求：SOP分正常流程和异常流程两部分，每步说明操作人、操作内容、完成标准、异常处理方式。格式适合打印张贴。
```

## 生活场景（31–40）

### 31 · 旅行计划

```
你是一位资深旅行规划师。帮我规划一份5天4晚的杭州旅行计划。

背景：我是第一次去杭州，两人出行，预算人均3000元，偏好文化体验和自然风光，不喜欢网红打卡点，6月中旬出行。

要求：每天一个大主题（避免行程碎片化），包含景点、交通、餐厅推荐（给出人均消费估算），附"行前打包清单"。
```

### 32 · 装修方案评估

```
你是一位室内设计顾问。帮我评估三套装修方案，给出选择建议。

背景：我家建筑面积98平米，三室一厅，预算硬装18万。方案A：现代简约风，设计感强但收纳空间少；方案B：日式原木风，收纳设计好但采光要求高；方案C：北欧风，风格安全但缺乏个性。

要求：从实用性、预算控制、长期居住舒适度三个维度评分（每项10分），给出综合推荐，并说明推荐理由。
```

### 33 · 孩子作文辅导

```
你是一位小学语文老师。帮我给10岁孩子辅导一篇描写小动物的作文。

背景：孩子想写家里的小猫，特点是毛是橘色的、喜欢趴在窗台上晒太阳、会用爪子洗脸。孩子写了一段但感觉写得像说明书，不生动。

要求：指出这段文字的问题在哪里（具体句子具体分析），给出修改方向，然后示范改写前3句，最后引导孩子思考"怎么让读者也喜欢上这只小猫"。
```

### 34 · 健康饮食建议

```
你是一位营养师。帮我设计一份适合上班族的一周晚餐食谱。

背景：我每天早8点到晚7点在公司，午餐在食堂吃，早餐简单，周末有时间可以备餐。身体情况：轻度缺铁，想控制晚餐碳水摄入，希望一周不重样。

要求：每天晚餐一份主菜+一份蔬菜+一份主食（碳水控制在100g以内），给出具体食材和做法简述。每周总热量估算在600-700大卡之间。
```

### 35 · 礼物挑选

```
你是一位礼品顾问。帮我在300元预算内挑选一份礼物。

## 学习场景（41–50）

### 41 · 英语学习计划

```
你是一位英语学习规划师。帮我设计一份为期8周的口语提升计划。

背景：我的英语基础是CET-6，读写还行但口语很差，和外国人开会只能说几个词。想在8周后能够用英语独立完成30分钟的商务电话。

要求：每周3个学习日，每天学习内容不超过40分钟，包含：听力输入（材料+时长）、输出练习（方法+话题）、1个重点句型。每周有明确的小目标。
```

### 42 · 论文润色

```
你是一位学术英语编辑。帮我润色以下论文摘要，使之更符合国际期刊发表标准。

背景：这是一篇关于"社交媒体使用对大学生睡眠质量影响"的实证研究摘要，目标期刊是《Journal of Health Psychology》（SSCI）。

要求：润色时不改变原意，重点改善：学术表达的精准性、句式多样性、时态一致性、术语规范性。润色后附上2-3条修改说明。
```

### 43 · 知识点讲解

```
你是一位高中数学老师。帮我用通俗的方式讲解"导数"这个概念。

背景：我的学生刚上高一，对函数有一定了解但没有接触过极限概念，数学基础中等，习惯用具体例子来理解抽象概念。

要求：讲解分三步：1）用生活中的例子引出"变化率"的概念（不要直接说"导数"）；2）用这个例子解释导数在解决什么问题；3）给出标准定义。避免出现"极限"、"微分"等术语。
```

### 44 · 考试复习计划

```
你是一位学习规划师。帮我制定一份期末考试复习计划。

背景：我是大一学生，有四门考试：高等数学（第21周考）、大学英语（第22周考）、经济学原理（第22周考）、计算机基础（第23周考）。距第一门考试还有5周，每周可投入学习时间约20小时。

要求：5周分配方案，给出每周各科复习重点、每周总时长分配、每日时间表示例。最后附一条复习效率建议。
```

### 45 · 读书笔记框架

要求：模板包含三个部分：1）读前提问（3个问题）；2）读中笔记（记录什么、怎么记）；3）读后总结（用自己的话复述、应用计划）。每个部分说明填写目的和具体操作方法。
```

要求：分为5个阶段，给出每个阶段的学习目标、推荐资源（免费为主）、练习项目。强调"每学一个概念就做一个实际任务"，不要堆知识点。

### 47 · 演讲准备

要求：帮我设计演讲框架，包含开场（1分钟）、正文三个模块、结尾（1分钟）。每个模块给出要讲的要点、建议用图还是文字、预计时长。

### 48 · 思维模型应用

```
你是一位商业思维导师。帮我用"第二层思维"分析一个投资决策。

背景：某公司刚发布财报，营收增长20%，股价却下跌了8%。用"第一层思维"（好财报→股价涨）无法解释这个现象。

要求：用第二层思维分三步分析：1）为什么股价会跌（列出至少3个第一层思维忽略的因素）；2）市场在担心什么；3）如果你的判断和市场相反，支撑你判断的核心理由是什么。
```

### 49 · 语言翻译与文化注释

要求：先给出英文翻译（语气专业但不失礼貌），然后解释为什么这样翻译，最后附3条中美商务沟通文化差异小提示。
