## 📋 合同类型

### 全职员工 vs 合同工

| 类型 | 说明 | 优点 | 缺点 |
|------|------|------|------|
| **全职员工 (W-2)** | 正式雇员，受劳动法保护 | 福利、保险、带薪假 | 雇主通常要求在特定地区 |
| **合同工 (1099)** | 独立承包商，自负税务 | 地点自由，可多项目 | 无福利，需自缴税，无保障 |
| **自由职业者** | 类似合同工，可通过平台接单 | 灵活度高 | 收入不稳定 |

### 合同工注意事项

- **1099 合同**：美国公司可合法雇佣远程合同工
- 合同工需自行申报收入、缴纳自雇税
- 确认合同中的工作范围、付款条款、知识产权归属
- 建议保留工作记录和沟通证据

## ⚠️ 法律注意事项

### 常见法律问题

| 问题 | 说明 |
|------|------|
| **工作许可** | 并非所有国家允许持旅游签远程工作，部分国家有数字游民专项签证 |
| **税务居民** | 各国税务规则不同，在一国停留超过183天可能成为税务居民 |
| **数据合规** | GDPR、CCPA 等数据保护法规，公司需确保合规 |
| **劳动法** | 合同工与全职员工适用不同的劳动法保护 |
| **知识产权** | 合同中需明确约定代码/作品的知识产权归属 |
| **竞业禁止** | 部分国家限制竞业条款，需注意合同约定 |

### 税务建议

- 了解自己成为税务居民的国家/地区的税务义务
- 考虑双重税务协定避免重复征税
- 合同工建议预留 25-35% 收入用于缴税
- 建议咨询专业国际税务顾问

## 💡 财务建议

### 自由职业者财务管理

1. **分离账户**：开设独立商业银行账户用于业务收支
2. **预留税款**：每笔收入预留 25-35% 缴税
3. **记录支出**：保留业务相关支出的发票和收据
4. **退休储蓄**：自行规划 401k 或 IRA
5. **健康保险**：自主择业者需自行购买保险
6. **收入稳定化**：建立 3-6 个月应急储备

### 远程工作 vs 传统工作对比

| 项目 | 远程全职 | 传统全职 |
|------|---------|---------|
| 薪资 | 通常相同或略高 | 标准 |
| 福利 | 可能减少（视公司） | 完整福利包 |
| 通勤 | 无 | 每天 0.5-2h |
| 住房 | 自理 | 自理 |
| 设备 | 公司可能提供 | 公司通常提供 |
| 社交 | 需主动建立 | 办公室自然社交 |
